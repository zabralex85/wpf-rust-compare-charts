# .NET WPF + Blazor Hybrid UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the WPF + Blazor Hybrid dashboard that replays `ride.db` in-process and renders the same dashboard as the Rust app — param table, two strip charts (ScottPlot.Blazor), gauges, GPS map (Leaflet via JS interop), perf HUD — matching `docs/reference/dashboard-target.md`.

**Architecture:** A `net8.0-windows` WPF app `TelemetryPoc.App` hosts a `BlazorWebView`. A `ReplayPlayer` (added to the green-tested `TelemetryPoc.Core` lib) reads samples via `TelemetryDb`, paces with `Pacer`, and feeds `TelemetryStore` on a timer — **no WebSocket** (idiomatic Blazor Hybrid, in-process). Razor components read the store and render; a render timer triggers `StateHasChanged`. This deliberately mirrors the Rust UI so the two are comparable; the only architectural difference is transport (in-process push vs. WebSocket), which is the realistic per-stack idiom.

**Tech Stack:** .NET 8 (`net8.0-windows`), WPF, `Microsoft.AspNetCore.Components.WebView.Wpf` (Blazor Hybrid), ScottPlot.Blazor (v5), Leaflet (JS interop), `TelemetryPoc.Core` (merged). xUnit for the `ReplayPlayer`.

## Global Constraints

- Consume the merged `TelemetryPoc.Core` (do NOT reimplement): `TelemetryDb`, `Pacer`, `TelemetryStore`, `ChannelSeries`, `ValueFormat`, `MetricsSampler`, models.
- In-process replay: a timer calls `player.Advance(elapsedMs, nowUnixMs)`; latency = `now - store.LastEmitUnixMs`.
- Channel→widget routing by `ChannelMeta.Widget` (`strip`/`gauge`/`table`/`map_lat`/`map_lon`); the param table shows ALL channels. Enum severity via `ValueFormat.SeverityColor(...)`.
- Strip x-axis in **seconds** (`TsMs / 1000`); window already enforced by the store's 60s `ChannelSeries`.
- DB path + replay speed configurable (env `RIDE_DB` default `../../data/ride.db` relative to the app exe, `RIDE_SPEED` default 1.0); fixture `data/ride_small.db` for quick runs.
- Layout + acceptance per `docs/reference/dashboard-target.md`.
- Nullable enabled; no warnings.
- ScottPlot.Blazor / BlazorWebView / Leaflet-interop APIs: the code below is the intended shape — **adapt to the actual installed package API where it differs** (note adaptations in the task report). Tasks 1, 3–9 are **build-verify + manual** (GUI); only Task 2 (`ReplayPlayer`) is unit-tested.

```csharp
// dotnet/src/TelemetryPoc.Core/ReplayPlayer.cs
namespace TelemetryPoc.Core;

public sealed class ReplayPlayer
{
    private readonly IReadOnlyList<Sample> _samples;
    private readonly TelemetryStore _store;
    private readonly Pacer _pacer;
    private int _next;

    public ReplayPlayer(IReadOnlyList<Sample> samples, TelemetryStore store, double speed)
    {
        _samples = samples;
        _store = store;
        _pacer = new Pacer(speed);
    }

    public bool Done => _next >= _samples.Count;

    public int Advance(long elapsedMs, long nowUnixMs)
    {
        int applied = 0;
        while (_next < _samples.Count && _pacer.DueOffsetMs(_samples[_next].TsMs) <= elapsedMs)
        {
            _store.ApplyFrame(_samples[_next], nowUnixMs);
            _next++;
            applied++;
        }
        return applied;
    }
}
```

```javascript
// dotnet/src/TelemetryPoc.App/wwwroot/gpsmap.js
window.gpsMap = (function () {
  let map = null, line = null, marker = null;
  return {
    init: function (el) {
      map = L.map(el, { zoomControl: false }).setView([32.08, 34.78], 11);
      L.tileLayer("https://tile.openstreetmap.org/{z}/{x}/{y}.png",
        { attribution: "© OpenStreetMap contributors", maxZoom: 19 }).addTo(map);
      line = L.polyline([], { color: "#e33", weight: 2 }).addTo(map);
      marker = L.circleMarker([32.08, 34.78], { radius: 5, color: "#fff", fillColor: "#e33", fillOpacity: 1 }).addTo(map);
    },
    update: function (lat, lon) {
      if (!map || lat.length === 0) return;
      const pts = lat.map((la, i) => [la, lon[i]]);
      line.setLatLngs(pts);
      const last = pts[pts.length - 1];
      marker.setLatLng(last);
      map.panTo(last, { animate: false });
    }
  };
})();
```

```razor
        await Task.CompletedTask;
    }
}
```

```razor
        var dbPath = Environment.GetEnvironmentVariable("RIDE_DB") ?? "../../data/ride.db";
        var speed = double.TryParse(Environment.GetEnvironmentVariable("RIDE_SPEED"), out var s) ? s : 1.0;
        using var conn = new SqliteConnection($"Data Source={dbPath};Mode=ReadOnly");
        conn.Open();
        var channels = TelemetryDb.LoadChannels(conn);
        var enums = TelemetryDb.LoadEnumValues(conn);
        var samples = TelemetryDb.LoadSamples(conn, channels);
        _store.ApplyMeta(channels, enums);
        _strip = channels.Where(c => c.Widget == "strip").ToList();
        _accZ = _strip.FirstOrDefault(c => c.ColumnName == "acc_z");
        _gauges = channels.Where(c => c.Widget == "gauge").ToList();
        _player = new ReplayPlayer(samples, _store, speed);

        _sw.Start();
        _timer = new System.Timers.Timer(33); // ~30 fps
        _timer.Elapsed += (_, _) => Tick();
        _timer.AutoReset = true;
        _timer.Start();
    }

    private long NowMs() => DateTimeOffset.UtcNow.ToUnixTimeMilliseconds();

    private void Tick()
    {
        var elapsed = _sw.ElapsedMilliseconds;
        _frameMs = elapsed - _lastTick;
        _fps = _frameMs > 0 ? 1000.0 / _frameMs : 0;
        _lastTick = elapsed;

        _player?.Advance(elapsed, NowMs());
        if (_store.LastEmitUnixMs > 0)
        {
            var sec = _store.LastEmitUnixMs / 1000; // ride-second cadence for metrics
            if (sec != _lastMetricSec) { _lastMetricSec = sec; _store.ApplyMetrics(_metrics.Sample()); }
        }
        InvokeAsync(async () =>
        {
            _main?.RefreshData();
            _bottom?.RefreshData();
            if (_map is not null) await _map.UpdateTrack();
            StateHasChanged();
        });
    }

    private IReadOnlyList<StripChart.StripLine> MainLines()
        => _strip.Where(c => c != _accZ).Select(ToLine).ToList();
    private IReadOnlyList<StripChart.StripLine> AccZLines()
        => _accZ is null ? Array.Empty<StripChart.StripLine>() : new[] { ToLine(_accZ) };

    private StripChart.StripLine ToLine(ChannelMeta c)
    {
        var s = _store.Series(c.Id);
        var (xs, ys) = s?.Arrays() ?? (Array.Empty<long>(), Array.Empty<double>());
        return new StripChart.StripLine(c.Name, xs, ys);
    }

    public void Dispose() { _timer?.Dispose(); }
}
```
